Talks and Materials for:

"An Introduction to Bioinformatics"

This Repository is for the Development and presentation of the Training Course entitled:

"A Web Based Introduction to Basic Bioinformatics"

It is comprised of four Branches:

  1. Devel: In which material is developed and maintained.
  2. master: Essentially a merge of the latest stable versions of Devel (which includes Data) and gh-pages.
  3. Data: Data and Software that should be downloaded in order for the Exercises to be efficiently worked through.
  4. gh_pages: In which the delivery web pages are developed and maintained.
